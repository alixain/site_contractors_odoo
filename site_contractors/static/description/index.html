What it does

This module provides a lightweight, standard approach to manage contractors and sites linked to a customer, and to pick them on Sales Orders.

  • Contractors are records linked to a Company (res.partner).
  • Each Contractor can have multiple Sites.
  • Sale Orders contain Contractor and Site fields with dynamic filtering.
  • List & Form views updated — search filters available.

Why it’s reliable

  • Separate models for Contractor & Site — no hack on parent/child contacts.
  • Proper access rights included.
  • Tested on Odoo Community.
Quick install
  1. Copy module folder to your addons path: /odoo/custom_addons/site_contractors/
  2. Restart Odoo and Update Apps list.
  3. Install Site Contractors from Apps.
  4. Create Contractors under Contacts → Contractors; add Sites under a Contractor; use in Sales → Orders.

Quick links

Helpful places in Odoo after install:

  • Contacts → Contractors
  • Sales → Orders (Contractor & Site fields)
  • Settings → Users & Companies (access rights)
